Sales Performance Dashboard (Power BI)
Overview
Interactive business dashboard built to monitor revenue, profit, growth trends, and product performance for data-driven decision-making.

Objective
Analyze sales data to identify growth trends, high-margin products, top-performing countries, and business improvement opportunities.

Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- MoM Growth
- Quantity Sold

Features
- Dynamic slicers (Date, Country, Product, Salesperson)
- Executive overview page
- Product performance analysis page
- Top N filtering
- Conditional formatting
- Time-series trend analysis

Key Insights
- 30% overall profit margin maintained
- Strong mid-year sales growth
- Revenue concentrated in top-performing products
- Opportunities for margin optimization

Tools Used
- Power BI
- DAX
- Data Modeling
